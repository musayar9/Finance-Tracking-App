# Finance Tracking App

## Project Description

This project is a modern mobile finance management application. It allows users to easily track their income, expenses, investments, and budgets. The app features a sleek dark mode design, offering a stylish and user-friendly experience.

## Purpose

To manage financial data in a single place, analyze spending habits, monitor investment portfolios, and achieve budget goals.

## Key Features

- **Dashboard (Home):** Displays total net worth, quick action buttons (Add, Transfer, Pay), and recent transactions.
- **Transactions**: Comprehensive list of all financial transactions with category filtering, date grouping, and color-coded income/expenses.
- **Portfolio**: Shows portfolio value, daily profit/loss, stocks and crypto assets, and detailed holding information.
- **Budget**: Monthly budget tracking, category-based spending analysis, visual progress bars.
- **Profile**: User information, security settings (Face ID, notifications), account settings.
- 
## Technologies Used

- **React Native**: Mobile app development
- **Expo**: Fast development and testing environment
- **TypeScript**: Type safety
- **Ionicons**: Vector icons
- **react-native-chart-kit**: Charts and graphs
- **Expo Router**: Navigation system

## Design System

- Dark Mode theme (#0D0D0D background, #1A1A1A cards)
- Glassmorphism effects
- Modern UI: soft shadows, large border radius
- Colors: Primary Green (#00E676), Accent Blue (#2979FF)

Installation & Running

1. Install dependencies:

   ```
   npm install
   ```

2. Start the app:
   ```
   npx expo start
   ```

## Folder Structure

- `app/(tabs)/`: Main screens (home, transactions, portfolio, budget, menu)
- `components/`: Reusable components
- `utils/`: Theme and global styles
- `constants/`: Mock data
- `types/`: TypeScript definitions

This application serves as a ready-to-develop UI Kit for modern fintech apps, covering all essential functionalities.
